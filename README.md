# Orbit-Radio
OrbitRadio is a UHF APRS telemetry payload from New Leap Labs, hosted on MOI-1. It transmits satellite health data at 435.248 MHz using AFSK/FM and AX.25 with 1 W power. Built for the global HAM community, it acts as a space-based digipeater, enabling real-time telemetry access and supporting education and research.
